4. **Temperature**: The temperature recorded for the given date and location. You can use a `FLOAT` or `DECIMAL` data type to represent temperatures.

5. **Precipitation**: The amount of precipitation recorded for the given date and location. This could be stored as a `FLOAT` or `DECIMAL` data type.

6. **Humidity**: The humidity level recorded for the given date and location. This could be stored as a `FLOAT` or `DECIMAL` data type.

7. **Weather Condition**: A description of the weather condition (e.g., sunny, cloudy, rainy). This could be stored as a `VARCHAR` data type.

8. **Wind Speed**: The wind speed recorded for the given date and location. You can use a `FLOAT` or `DECIMAL` data type to represent wind speeds.

9. **Weather Events**: Any specific weather events recorded for the given date and location (e.g., thunderstorm, snowstorm). This could be stored as a `VARCHAR` data type.

10. **Last Updated**: The date and time when the weather data was last updated. You can use a `DATETIME` or `TIMESTAMP` data type for this.

    WindSpeed FLOAT,
